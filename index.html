Taxonomía del Dataset

Clasificación Taxonómica Real

Las especies del dataset pertenecen a 4 familias, 7 géneros y 13 especies registradas en campo. La anotación en Label Studio sigue esta jerarquía exacta: Familia → Género → Especie.

Objetivo del Proyecto

Identificación de Anura con IA de última generación

Construir un sistema integral de visión por computadora que identifique especies de ranas a nivel taxonómico completo (familia → género → especie), segmente el cuerpo sobre el fondo e identifique partes anatómicas específicas con precisión a nivel de píxel. El sistema integra coordenadas GPS para contextualización en biodiversidad.

🔬

Clasificación Taxonómica

Predicción simultánea de Familia, Género y Especie mediante un modelo multi-tarea con fine-tuning sobre BioCLIP, pre-entrenado en datos biológicos.

🎭

Segmentación Binaria

U-Net con backbone ResNet-34 para separar la rana del fondo con máscara binaria pixel-exacta. Incluye augmentación geométrica y fotométrica intensiva.

🧬

Anatomía Semántica

SegFormer fine-tuned identifica 8 partes anatómicas: cuerpo, ojos, manos, patas, tímpano, boca y piel — útil en investigación de campo y laboratorio.

📍

Geolocalización

Integración de coordenadas GPS en formato JSON por imagen, con conexión opcional a GBIF para enriquecer los registros de ocurrencia de biodiversidad.

🛡️

Falsos Positivos

El dataset incluye imágenes de objetos similares a ranas y fondos confusos para entrenar robustez ante condiciones reales de campo.

⚡

API REST

Endpoint FastAPI /predict que acepta imagen y devuelve JSON con especie, máscara, mapa anatómico y coordenadas.

Modelo de Segmentación Binaria

Pipeline de Máscara Binaria

Las imágenes PNG con canal Alpha (RGBA) se convierten a máscara binaria uint8. El canal Alpha define la transparencia: Alpha > 128 → rana (1), Alpha ≤ 128 → fondo (0). La imagen original (con fondo) y su máscara van juntas al modelo U-Net.

División del Dataset · Segmentación Binaria

Albumentations · Pipeline

Pipeline de Augmentación

Con ~1,100 imágenes reales la augmentación no es opcional: es el factor que determina si los modelos generalizan o fallan. Todas las transformaciones geométricas se aplican simultáneamente a la imagen y su máscara para mantener correspondencia pixel-exacta y evitar overfitting.

⚠️

Eliminación de Fondo — Clave Anti-Overfitting

El modelo de segmentación binaria aprende a ignorar el fondo en lugar de memorizarlo. Las imágenes originales incluyen el fondo real de campo — el nombre de la imagen original debe coincidir con el de su máscara. CutMix entre imágenes de ranas y fondos genera ejemplos difíciles (falsos positivos). El CoarseDropout simula oclusiones parciales del cuerpo para evitar que el modelo dependa de la rana completa.

Métricas

Evaluación por Módulo

Cada modelo se evalúa y entrena de forma independiente. El test set nunca es visto durante el entrenamiento. Se reportan métricas por clase para detectar fallos en clases clave.

Módulo Métrica Principal Métrica Secundaria
Clasificación especie F1-score macro Top-3 Accuracy
Clasificación género F1-score macro Confusion Matrix
Clasificación familia F1-score macro Precision / Recall
Segmentación binaria IoU (Jaccard) Dice Score
Segmentación semántica Mean IoU (mIoU) IoU por clase

Diseño del Sistema

Decisiones Clave del Proyecto

Cada decisión de diseño está fundamentada en las características del dataset y los objetivos de investigación.

✓

Sin edición artificial

Los datos reales de campo son la ventaja diferencial del proyecto.

✓

Falsos positivos en training

Robustez ante fondos confusos y objetos similares a ranas.

✓

Pipeline modular (3 modelos)

Cada tarea tiene su complejidad propia e independiente.

✓

Fine-tuning con backbone pre-entrenado

Dataset pequeño: transfer learning es esencial para generalizar.

✓

Taxonomía jerárquica

Familia → Género → Especie: coherencia biológica en predicciones.

✓

Segmentación semántica (no bounding boxes)

Precisión anatómica a nivel de píxel para investigación científica.

✓

Dataset propio etiquetado

Ventaja competitiva y control de calidad total sobre los datos.

✓

Evaluación honesta test set
